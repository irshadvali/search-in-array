# search-in-array
#How to use
```
  import SEARCHARRAY from "search-in-array";


  let data=[{id:1, name:"irshad"},{id:2, name:"vali"},{id:3, name:"suresh"}]
  let aaa= SEARCHARRAY.getSearchResult("ir",data,"name")
  console.log(aaa);
  
  /*
   getSearchResult(val, data, searchByNode)
   Where:
   val: this is input value.
   data: Array or list.
   name: this is node of array
  */
```
